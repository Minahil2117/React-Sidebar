# SaaS Admin Dashboard Sidebar (React + Vite)

A modern, responsive, and animated SaaS-style admin dashboard sidebar built using React and Vite.

This project includes:

- Collapsible sidebar
- Smooth animations
- Dropdown sub-menus
- Active menu highlighting
- React Icons integration
- Professional SaaS UI design

---

## Tech Stack

- React (Vite)
- JavaScript (ES6)
- CSS3
- React Icons

---

## Features

### 1. Collapsible Sidebar
The sidebar can expand and collapse with smooth width transition.

### 2. Sub Menu Options
Menu items like Analytics, Users, and Settings contain expandable sub-options.

### 3. Active State Highlight
The selected menu and sub-menu item is highlighted.

### 4. Smooth Animation
Dropdown submenus animate smoothly using CSS keyframes.

### 5. Modern SaaS UI
Dark sidebar theme with clean spacing and professional styling.
